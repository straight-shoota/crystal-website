    Crystal Ecosystem

    The Crystal ecosystem is backed by libraries called shards, many are available as Open Source software.

    Dependencies are managed by the eponymous program shards which is included in most Crystal distributions.

    Shard Indices

    Shards are a decentralized system and there is no central registry. Instead, there are indices which allow you discover shards.

    Shardbox.org

    A curated database of Crystal shards. It helps developers to discover shards and the Shards ecosystem.

    shards.info

    Lists shard repositories on GitHub and GitLab with dependencies and releases.

    crystalshards.org

    Lists shard repositories on GitHub with dependencies.

    awesome-crystal

    A curated Awesome List of Crystal shards, tools and tutorials.
